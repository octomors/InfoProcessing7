# InfoProcessing7

Сайт для GitHub Pages: загрузка изображения и бинаризация двумя способами.

## Возможности

- Ручная бинаризация по порогу (ползунок 0-255).
- Автоматический порог методом Otsu.
- Предпросмотр исходного и бинаризованного изображения.
- Скачивание результата в PNG.

## Запуск локально

Откройте `index.html` в браузере.

## Публикация на GitHub Pages

1. Запушьте содержимое репозитория в GitHub.
2. В настройках репозитория откройте раздел Pages.
3. Выберите ветку (обычно `main`) и корневую папку (`/root`).
4. Сохраните настройки и дождитесь публикации.
